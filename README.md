word animate 

## Install
```
npm i @simon_he/animate-text
```

## Usage
```js
import animateText from '@simon_he/animate-text'
animateText({
  text: '大家好,我是Simon;来自上海;我热爱coding'.split(';'), // string or string[]
  infinity: true, // infinite loop
  container: '#app', // container selector
})
```
